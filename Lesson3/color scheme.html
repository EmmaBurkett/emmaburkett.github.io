Color scheme

The website will utilize many of the colors displayed on this page, with the main color being a darker blue.
The darker blue will be used on items that donâ€™t change with day to day updates, for example:
The title will not change color. It will also be used as a border to some items, like around the forecast.
This color will be followed up by an orange-yellow color which will be used to highlight buttons, and as a common theme around navigational tools and links.
An off-white will be used as the main background of the website, and a gray will be used as the font color.
Other colors that will be featured in the website is a fern green, and some lighter shades of green. This will be used as a common theme in pictures, that would later be replaced with ads.

Colors in RGB format:

Dark Blue: rgb(19, 105, 180)
Orange-Yellow: rgb(255, 208, 0)
Off-white: rgb(245, 245, 245)
Grey: rgb(211, 211, 211)
Fern Green: rgb(88, 132, 67)
