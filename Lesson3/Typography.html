Colors and Size

The Main Heading: 'Dancing Script', cursive, will be mainly used as a title font for each page. It will be large, to make it more readable. It will be colored the dark blue referanced in the color scheme page.
The Secondary Heading: 'Shadows Into Light', cursive, will be used mainly for sub headings found througout a page and the navigation bar. It will vary between dark blue and fern green.
The main text: 'Dosis', sans-serif, will be used for any large body of text. It will be colored grey, to avoid stain on readers eyes.
